Описание

Строительство жилого дома ведется в Заводском районе г. Орла по адресу: ул. Розы Люксембург д.33
Жилой дом запроектирован односекционным 16 этажей. Проектом предусмотрены 1-но, 2-х, 3-х комнатные квартиры. Общее количество квартир в жилом доме — 159.
Проектом предусмотрены: детская площадка для подвижных игр и спорта, площадка для отдыха, хозяйственная площадка. Площадки оборудованы малыми архитектурными формами. Предусмотрены гостевые стоянки для автомобилей.
